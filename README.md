# Daniel Darzi's Analytics Portfolio
## Bellabeat-Wellness-Analysis Case Study
Analytics portfolio – Bellabeat wellness dataset exploration, cleaning, and visual analysis.

**Tools used:** Excel, data cleaning, scatterplots, categorical charts, data storytelling

###Project Overview
This project analyzes wellness‑related data from Bellabeat to explore patterns in daily activity, sleep, steps, calories, and other health indicators.
The goal is to identify meaningful relationships between variables and communicate insights through clear, professional visuals and narrative analysis.

###What I Did
- Cleaned and prepared the wellness dataset for analysis
- Created multiple scatterplots with trendlines and R² values to examine correlations
- Built categorical visuals to compare patterns across groups
- Engineered helper columns and ensured ISO‑standard formatting for audit‑ready workflow
- Integrated visuals into a polished final report with clear explanations and insights
